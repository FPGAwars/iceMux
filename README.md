## Install

For installing and using this colection in Icestudio follow these steps:

* Download the collection: [stable](https://github.com/FPGAwars/iceMux/archive/refs/tags/v0.1.0.zip) or [development](https://github.com/FPGAwars/iceMux/archive/refs/heads/master.zip)
* Install the collection: *Tools > Collections > Add*
* Select the collection: *Select > Collection*
